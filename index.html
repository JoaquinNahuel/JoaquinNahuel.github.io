Proyectos universitarios

PROYECTO CINE

En este proyecto se realizó una aplicación de escritorio desarrollada con PyQt6, un conjunto de herramientas para crear interfaces gráficas en Python. El proyecto se centra en un sistema para la gestión de un cine, que incluye funciones para la compra de boletos y alimentos, así como la administración de precios y productos. La interfaz consta de varias ventanas:

    Cine_Menu: Presenta opciones para acceder como cliente o administrador, así como para comprar alimentos.

    Cliente_Comida: Permite al cliente seleccionar alimentos y bebidas para comprar. Admin: Ofrece funcionalidades de administración, como la gestión de precios y productos de alimentos y bebidas.

    Cliente: Permite al cliente comprar boletos para las salas de cine, con opciones para diferentes tipos de boletos.

     Sala1 y Sala2: Ventanas que simulan la selección de asientos en diferentes salas de cine.

    El programa facilita la interacción entre los clientes y el personal del cine, brindando una experiencia intuitiva y completa para la gestión de compras y ventas en un entorno de cine.
